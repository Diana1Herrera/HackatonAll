# 🏀 Tienda Deportiva Infantil - Landing Page Interactiva 🛹

¡Bienvenidos a nuestra **Landing Page Interactiva** de una Tienda Deportiva Infantil! Este proyecto fue diseñado y desarrollado en equipo para ofrecer una experiencia visual y funcional única, mostrando la magia del deporte en los más pequeños. 🧒⚽🎾

## 🌟 Características Principales
- **Interfaz Atractiva y Adaptable:** 
  Utilizamos **HTML** y **CSS** (con **Bootstrap**) para crear un diseño limpio, moderno y  responsivo que luce genial en cualquier dispositivo.
  
- **Animaciones Interactivas:** 
  Gracias a **JavaScript**, nuestra página incluye:
  - Transiciones dinámicas para mejorar la experiencia del usuario.
  
- **Catálogo Deportivo:** 
  Explora una gran variedad de productos diseñados especialmente para los pequeños deportistas.

- **Enfoque Profesional:** 
  Este proyecto fue desarrollado con **Visual Studio Code**, integrando de forma eficiente todas las herramientas y lenguajes utilizados.

## 🛠️ Tecnologías Utilizadas
- **HTML**: Para la estructura de la página.
- **CSS / Bootstrap**: Para estilos modernos y responsivos.
- **JavaScript**: Para animaciones y funcionalidades interactivas.
- **Visual Studio Code**: Nuestro entorno de desarrollo.

## 🎯 Objetivo del Proyecto
El objetivo fue crear una landing page que no solo destaque visualmente, sino que también brinde una experiencia interactiva para quienes la visitan. Este proyecto forma parte de mi portafolio y refleja nuestras habilidades como equipo para combinar creatividad y tecnología.
